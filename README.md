# Seif Portfolio

A modern, dynamic personal portfolio built with **Angular** and powered by **Google Sheets** as a lightweight CMS. This setup allows for real-time content updates without the need for code changes or redeployments.

[**Live Demo**](https://seifeldeen-portfolio.netlify.app/home)

---

##  Key Features

* ** Dynamic Content Management:** Integrated with **Google Sheets API**. I can update my projects, skills, and bio directly from a spreadsheet, and the portfolio reflects changes instantly.
* ** Integrated Contact System:** A fully functional contact form that allows visitors to send emails directly to my inbox.
* ** Fully Responsive:** Optimized for all screen sizes, from mobile to desktop.
* ** Fast Deployment:** Hosted on **Netlify** for high performance and reliability.

---

##  Tech Stack

* **Frontend:** [Angular](https://angular.io/)
* **Database / CMS:** [Google Sheets API](https://developers.google.com/sheets/api)
* **Styling:** CSS3 
* **Hosting:** [Netlify](https://www.netlify.com/)

---

##  How it Works

The project uses a custom service to fetch JSON data from a published Google Sheet. This approach decouples the data from the UI, making it:
1.  **Maintenance-free:** No need to push code for small text changes.
2.  **Scalable:** Easy to add new projects or experience entries.
